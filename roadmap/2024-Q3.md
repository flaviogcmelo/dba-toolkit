## Dependencies / Risks

| Risk | Mitigation |
|------|-----------|
| Docker/Vagrant complexity | Start with Docker Compose, Vagrant later |
| Oracle license in labs | Use Oracle XE (free) for labs |
| Ansible role quality | Use `ansible-lint` in CI from day one |
| Shell portability | Run ShellCheck in CI |
